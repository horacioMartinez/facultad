Para realizar la instalacion se deben seguir los siguintes pasos:

1) Abrir una consola y movernos dentro del directorio root.
2) Ejecutar el siguiente comando:
./installer.sh <directorio_donde_queremos_instalarlo>

De no encontrarse el directorio, lo instalara en el directorio en el 
cual se encuentra, dentro de una carpeta default la cual se llamara 
install_dir.
